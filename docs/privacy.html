Personvernerklæring

Sist oppdatert: 02.02.2026

Denne personvernerklæringen beskriver hvordan Samlivsbrudd behandler personopplysninger. Målet er å gi en enkel oversikt over hva vi samler inn, hvorfor vi gjør det, og hvilke rettigheter du har.

1. Behandlingsansvarlig

ltj-production/ltj er behandlingsansvarlig for personopplysninger som behandles i forbindelse med tjenesten.

Kontakt: latajo@gmail.com · Bergen

2. Hva vi samler inn

Vi samler inn så lite som mulig. Tjenesten er laget for å fungere uten innlogging og uten sentral lagring av brukerdata.

Eventuelle data som kan forekomme:

  • Tekniske data fra nettleseren (f.eks. nettlesertype)
  • Loggdata fra server (f.eks. tidspunkt og IP-adresse) – hvis relevant

Ingenting lagres sentralt, alt lagres lokalt og kan fjernes med et tastetrykk.

3. Formål og rettslig grunnlag

Formålet med eventuell behandling er å levere tjenesten, forbedre stabilitet og sikkerhet, og sikre normal drift.

Rettslig grunnlag: berettiget interesse eller samtykke, avhengig av hva som behandles. [JUSTER VED BEHOV]

4. Deling med tredjeparter

Vi deler ikke personopplysninger med tredjeparter, med mindre dette er nødvendig for drift eller lovpålagt.

Ingen.

5. Lagringstid

Vi lagrer ikke brukerdata sentralt. Eventuelle tekniske logger lagres kun så lenge det er nødvendig for drift og sikkerhet.

6. Dine rettigheter

Du har rett til innsyn, retting, sletting, begrensning og å protestere mot behandlingen der dette er aktuelt. Du kan også klage til Datatilsynet.

7. Endringer

Vi kan oppdatere denne erklæringen ved behov. Dato for siste oppdatering står øverst.
